## 바닐라 JS 챌린지

> 배열 합차기 / 오브젝트 합치기
- 변수를 만들 때 순서가 있다.
  먼저 변수를 생성 ->초기화 -> 사용
  ```javascript
  let a = 221;
  b = a - 5;```
- 변수를 초기화할 때나 생성할 때
let, var, const를 사용한다.
- const : 상수(안정적) : 정의하면 값을 바꿀 수 없다)
- var : let처럼 값을 바꿀 수도 있어서 사용하지 않는다.

> Array
- 값을 나열하는것. 아무거나 들어올 수 있다.
- value1, value2, value3, ... 이런 형식이다.
- value1을 호출하고 싶다면 array이름[숫자]로 호출 (숫자는 0부터)
- 안에 value를 넣고 싶다면 push(),concat()같은 명령어를 사용하면 된다.

> Object
- Array에서 자료를 빼올 때마다 호출하는 숫자를 기억해야되서 그 숫자를 key로 만듬
- key : value 형식
- object이름.key이름 으로 호출
- value를 바꾸고 싶으면 object이름.key이름 = '변경값'
- 오브젝트끼리 합치기(merge) = 
ObJect.assign(원래,출처)
- 오브젝트 복사 = 
ObJect.assign({},출처)


Object -> String JSON.stringify  
String -> Object JSON.parse
