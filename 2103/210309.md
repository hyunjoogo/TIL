# 새로운 페이지를 맞는 자세

이 페이지의 역할은 무엇이고 각 창, 하나의 인풋이라도 의미를 파악하면
어느 곳에서 어떤 것을 해야할지 보이는 것 같다.

순서를 정하는 것도 중요한 것 같다.
순차적으로 하되 걸리는 부분이 있다면 (기능 구현, 이슈 발생 등)
속 시원하게 넘어가자. 괜히 끙끙되지 말고.
한 파트 하기 전에 어떻게 할 것인지 적거나 노션에 남기고
