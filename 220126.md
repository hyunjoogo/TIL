### styled-components로 라이브러리 내 css 구현하기

클래스네임으로 스타일을 주는 라이브러리는 GlobalStyle을 만들어서 변경할 수 있다.

```css
export const Global = createGlobalStyle`
  .slick-slide {
    display: inline-block;
  }
  .ant-card-cover {
    transform: none !important;
  }
`
```
